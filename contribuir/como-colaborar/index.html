Como Colaborar com o Projeto Debian

Você quer começar a colaborar com o Debian? Veja abaixo onde seu perfil mais se encaixa. Se você é um(a) colaborador(a) e seu método ou grupo de colaboração não esteja listado nessa página, por favor compartilhe conosco!

  • Usar, testar, e enviar bugs para o Sistema de Rastreamento de Bugs(BTS - Bug Tracking System

Ajudar outros usuários em:

  • Listas de discussão oficiais;

  • Canais de IRC

  • Fórum;

  • Grupo no Telegram.

Traduzir

  • Descrições de pacotes;
  • Manpages;
  • Modelos debconf
  • Páginas web.
  • Instalador do Debian Mais informações acesse Wiki do Debian

Divulgar o Debian

  • Sites;
  • Escolas
  • Universidades/Faculdades;
  • Eventos como congressos, fóruns, encontros, etc.

Escrever tutoriais

Escrever tutoriais básicos ou avançados, para usuários ou desenvolvedores. A wiki do Debian é um bom lugar para iniciar.

Enviar testes, patches bugs e relatórios

  • Usar, testar, e enviar bugs para o Sistema de Rastreamento de Bugs(BTS - Bug Tracking System
  • Enviar patches com correções para bugs
  • Fazer testes de instalação e enviar relatórios.

Desenvolver atividades

  • Desenvolver atividades de publicidade e imprensa, e escrever notícias.
  • Organizar eventos e encontros focados em Debian na sua cidade.
